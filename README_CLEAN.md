# NSI Clean Room

Этот контур лежит в `C:\Curs\NSI_agent\nsi_clean` и предназначен для аккуратного восстановления проекта без риска сломать исходную смесь.

Что уже выровнено:

- единый импорт иерархии и моделей;
- синхронный режим без зависимости от Celery для базового сценария;
- согласованные endpoints для PPR;
- исправленный экспорт иерархии и общий экспорт;
- базовый pipeline `документ -> разбор -> характеристики модели`;
- UI-кнопки для разбора паспорта и автозаполнения характеристик.

Ограничения текущего этапа:

- парсер паспорта сейчас базовый и работает по текстовым/HTML-подобным документам;
- PDF/OCR не реализованы как полноценный промышленный контур;
- часть продвинутой логики ТК/спецификаций пока использует существующий каркас.

Рекомендуемая следующая итерация:

1. Добавить полноценный extractor PDF/OCR.
2. Довести массовую обработку моделей до пакетного конвейера.
3. Вынести техкарту/спецификации в отдельные сервисы генерации из документации.
4. Добавить интеграционные тесты clean-room контура.
